# XXConnect
Web-application with AI noise reduction


## Есть
- сигнальный сервер (принимает по id, отправляет по id, закрывается по Ctrl+C (или другим способом, подающим сигнал SIGINT, SIGTERM))
- менеджер соединений (принимает клиента по Id, позволяет создать PeerConnection (извне - как инициатор (отправляет оффер), внутри - принимает оффер, отправляет answer))
